## Dev

```shell
git clone --depth 1 https://github.com/oeyoews/tiddlywiki-codemirror6
cd tiddlywiki-codemirror6
pnpm install ## add dependencies
pnpm dev  ## local debug plugins
pnpm build  ## build codemirror6 plugins
```
